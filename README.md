# Application-web-pour-la-pr-diction-par-mod-le-de-classification

Application web développée avec Streamlit permettant de prédire le risque d’accorder un crédit à un client grâce à un modèle de classification en machine learning. 
L’application analyse les données clients pour fournir une estimation du risque (faible ou élevé) et aider à la prise de décision.
